[ANN]:

Training and testing data set downloaded from https://github.com/sausheong/gonn/blob/master/mnist_dataset/mnist.zip

Ensure that data/ dir is a subdir of Go-Projects/ann/

To run the ANN:

&nbsp;&nbsp;go build
    
&nbsp;&nbsp;./ann -mnist train # for running the ann on training data set
    
&nbsp;&nbsp;./ann -mnist predict # for running the ann on testing data set
    

[SERVER]:

To run the server:

&nbsp;&nbsp;go build 
&nbsp;&nbsp;run the executable(main) in the directory (Contest directory) in which you want to create your problem directory
